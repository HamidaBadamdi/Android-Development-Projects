## 📌 About This Repository

This repository contains my **Android development learning projects**, where all concepts are implemented inside a **single Android project** named **MyApplicationMCA**.

Instead of creating a new project for every topic, I follow a structured approach:

✅ One main project
✅ Each new Android concept → implemented using a **new Activity**
✅ Separate XML layouts and required resources added for every feature

## 📱 Project Structure

* **MyApplicationMCA**

  * New Activities for each concept
  * Corresponding XML layout files
  * Required Java/Kotlin and resource files

## 🚀 Purpose

This repository serves as:

* A centralized Android learning workspace
* Concept-wise practical implementation
* Continuous hands-on practice and experimentation

## 📚 Topics Covered

New concepts will be added progressively, including:

* Activity Lifecycle
* UI Components
* Intents & Navigation
* Layouts
* Event Handling
* Android Features & APIs

## 🔄 Updates

Future updates will **extend the same project** by adding:

* New Activities
* XML layouts
* Supporting files for new Android concepts

This approach helps maintain all learning progress in one scalable Android application.
